# FUTURE_DS_01 – Automobile Sales Dashboard

## Internship Task 1 – Data Visualization Dashboard

This project presents an interactive Power BI dashboard analyzing automobile sales data across different countries, models, and engine types.

## Dashboard Features

* KPI Metrics

  * Total Units Sold
  * Total Marketing Spend
  * Average Price
  * Revenue YoY Growth %

* Visualizations

  * Revenue Trend Over Time
  * Total Revenue by Model and Year
  * Revenue by Country

* Interactive Filters

  * Year
  * Country
  * Engine Type
  * Model

## Tools Used

* Power BI
* GitHub
